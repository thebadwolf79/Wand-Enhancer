# Security Policy

## Supported Versions

Only the latest released version of Wand-Enhancer receives security fixes.
Please upgrade to the most recent [release](https://github.com/thebadwolf79/Wand-Enhancer/releases)
before reporting an issue.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, report them privately through GitHub's
[Security Advisories](https://github.com/thebadwolf79/Wand-Enhancer/security/advisories/new)
("Report a vulnerability"). This keeps the details confidential until a fix is
available.

When reporting, please include:

- A description of the vulnerability and its impact
- Steps to reproduce (proof-of-concept if possible)
- The affected version(s)
- Any suggested remediation

You can expect an initial response within a few days. Once the issue is
confirmed, a fix will be prepared and released, and you will be credited in the
advisory unless you prefer to remain anonymous.

## Scope

Wand-Enhancer patches a third-party desktop application. Reports are most
valuable when they concern:

- The patcher modifying files outside its intended target
- Tampering with or spoofing the downloaded/extracted payloads
- The bundled web panel exposing unintended network surface
- Handling of credentials, tokens, or user data

Thank you for helping keep the project and its users safe.
